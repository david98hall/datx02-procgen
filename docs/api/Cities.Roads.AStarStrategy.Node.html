Class AStarStrategy.Node

Private class representing a node in a path. Is used by Path() for finding the optimal node between two points. Implements the System.IComparable<T> interface for a custom sorting order in sorted set.

CompareTo(AStarStrategy.Node)

Custom System.IComparable<T>.CompareTo(T) method for defining the sorting order between nodes. Sorting order is defined by the lowest sum of _cost and _heuristic. Equality is defined as equal locations, to avoid duplicate nodes with the same location in a System.Collections.Generic.SortedSet`1. Equal costs is defined as one being lower, to allow nodes with equal cost in a System.Collections.Generic.SortedSet`1.

Cost(AStarStrategy.Node, Single)

Calculates cost between two nodes. The cost is defined as the sum of the 3D distance between the node and the 3D distance times the height difference between the nodes. A height bias defined by _heightBias determines the impacts of the height difference. When the height bias is zero, the height has no impact.
