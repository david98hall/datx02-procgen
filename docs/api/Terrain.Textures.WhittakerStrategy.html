Class WhittakerStrategy

Generates 2D textures from a given height map with a simple version of a Whittaker diagram. A Whittaker diagram define a simple ecosystem depending on temperature and precipitation. This class also takes the height into consideration. see IGenerator<T>.

ComputeColor(Single, Single, Single)

Computes a color from a simple Whittaker ecosystem using a given height, precipitation and temperature.

The colors of the ecosystem is defined as: Whittaker [Precipitation, Temperature] in [0, 0] to [1, 1] Gray = [0, 0] White = [1, 0] Yellow = [0, 1] Green = [1, 1]

The color is computed by interpolated between the colors using the precipitation and temperature. The precipitation and temperature are recalculated with the height as a bias.
