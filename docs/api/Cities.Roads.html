Namespace Cities.Roads

Classes

AStarStrategy

Generates the optimal roads between a set of start and goal nodes with a given height bias. Extends the Strategy<TI, TO> class.

AStarStrategy.Node

Private class representing a node in a path. Is used by Path() for finding the optimal node between two points. Implements the System.IComparable<T> interface for a custom sorting order in sorted set.

Factory

Creates strategies for generating road networks.

LSystem

A way of generating roads using a non-deterministic L-system.

LSystem.State

Describes the position and direction of the L-system

LSystemStrategy

A strategy which is injected with a tuple of terrain data: its offset in the scene, and its height map. Based on this data, a road network can be generated with an L-system algorithm.

RoadNetwork

Represents a network of roads.
