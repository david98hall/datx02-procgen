Namespace BIAS.PCG.Cities.Buildings

Classes

Building

Generated buildings. Each building is defined by its vertices, bounds, position and facing.

ExtrusionStrategy

Generator for buildings. Will use strategy to construct building according to lot shape and population density (NYI). Some plots are suitable for multiple buildings; these should be split into street-bordering buildings and a central green area. This can be done by a Lot generator.

Factory

Creates strategies for generating buildings.

Lot

Contains exactly one building. Contains area, center and facing for building generation operations.
