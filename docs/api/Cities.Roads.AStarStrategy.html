_paths

A dictionary of the the start and goal nodes. Start nodes are given as keys. Goal nodes from a given start node is given as a set of nodes from the start key. Is used in Generate() to find the optimal path between each pair of start and goal nodes. Each pair of start and goal nodes is implicitly used in Path(AStarStrategy.Node, AStarStrategy.Node, Single[,]) to found the optimal path between the given pair of nodes. Since the height map in Injector may be changed externally, the nodes in the dictionary may be invalid.

Path(AStarStrategy.Node, AStarStrategy.Node, Single[,])

Find the optimal path by using the A* search algorithm between a start and goal node given as AStarStrategy.Node objects. The optimal path is defined as the cheapest path in terms the cost between the start and goal node. The cost is defined as the sum of the cost of reaching the node, _cost, and the estimated cost to the goal from the node, _heuristic, as given by the A* algorithm. The path with the lowest cost is found by using a System.Collections.Generic.SortedSet`1 object with a custom System.IComparable<T> object defined with CompareTo(AStarStrategy.Node). The search is expanded by using Adjacent(Single[,], ICollection<AStarStrategy.Node>, AStarStrategy.Node, Single) of the currently cheapest node.
